# Hotel Como - Taller de Programación Web

## Descripción
Este proyecto consiste en el desarrollo de un sitio web responsivo para el **Hotel Como**. [cite_start]El sitio ha sido construido utilizando HTML5, CSS3 y el framework Bootstrap, aplicando buenas prácticas de estructura y diseño para cumplir con los requisitos del Taller de Programación Web de la Universidad de Talca.

## Estructura de Carpetas
[cite_start]Para que el sitio se visualice correctamente se mantiene el siguiente orden de archivos[cite: 133]:
- `index.html`: Archivo principal del sitio.
- `styles.css`: Hoja de estilos personalizados.
- `img/`: Carpeta que contiene todas las imágenes locales utilizadas.
- `README.md`: Este archivo con instrucciones.
